# epb
Simple app to run pocketbase server in any folder

##### Clone or download this repository

##### **remove ".example" from ".env.example" and set the POCKET_DIR environment variable to your pocketbase folder. 

###### Running:
<pre>
<code>
    go run .
</code>
</pre>

###### Build:
<pre>
<code>
    ./build.sh
</code>
</pre>

##### Result:

###### directory  " build "
